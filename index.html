TRS-80 MODEL III

Your keyboard is the TRS-80's keyboard. Press ENTER at Cass? and Memory Size? to reach READY, then type BASIC. ESC is BREAK, Backspace deletes. Typing is Model III style: the machine boots in caps mode — press SHIFT+0 to unlock lowercase, then SHIFT+letter types lowercase. The machine case can be hidden from the MACHINE menu.
